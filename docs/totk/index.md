# 王國之淚 TotK

《薩爾達傳說：王國之淚》技巧與 Glitch 攻略。分類涵蓋虛化、儲存、左納烏裝置、跨檔繼承、速通錯傳等。每篇頁面都會標註**適用版本**與**難度**。

!!! tip "版本提醒"
    王國之淚常見版本：NS1 的 **1.2.1**、NS2 的 **1.4.3** 等。許多 glitch 只在特定版本可用，操作前務必確認頁面標註的適用版本。
